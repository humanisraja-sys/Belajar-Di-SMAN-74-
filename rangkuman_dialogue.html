Jendela Dialog Javascript

Interaksi Cepat Antara Web dan Pengguna

Javascript menyediakan tiga jenis jendela dialog bawaan yang berjalan di atas objek window. Dialog ini bersifat blocking, artinya eksekusi kode akan terhenti sampai user merespon.

!

1. Dialog Alert

Digunakan untuk menampilkan informasi atau peringatan satu arah. Hanya memiliki satu tombol: OK.

alert("Selamat Datang!");
?

2. Dialog Confirm

Digunakan untuk meminta konfirmasi (Ya/Tidak). Mengembalikan nilai true jika klik OK, dan false jika klik Cancel.

let yakin = confirm("Hapus data ini?");
if (yakin) {
    // eksekusi hapus
}
‚å®

3. Dialog Prompt

Digunakan untuk mengambil input teks dari user. Mengembalikan nilai berupa string atau null jika dibatalkan.

let nama = prompt("Siapa nama kamu?", "Guest");
document.write("Halo " + nama);
Tips: Meskipun mudah digunakan, dialog bawaan browser ini tidak bisa di-custom tampilannya secara visual. Untuk aplikasi modern, developer sering menggunakan library seperti SweetAlert2 atau modal custom HTML/CSS.
