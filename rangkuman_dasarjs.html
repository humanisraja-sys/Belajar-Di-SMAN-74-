Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang awalnya dibuat untuk membuat halaman web menjadi interaktif. Bersama HTML dan CSS, JS menjadi teknologi inti dalam pengembangan web.

Cara Menulis Kode JS

Ada dua cara utama untuk menyisipkan JavaScript ke dalam HTML:

  • Internal: Menggunakan tag <script> langsung di dalam file HTML.
  • Eksternal: Menulis kode di file terpisah dengan ekstensi .js lalu menghubungkannya dengan <script src="file.js">.
<script>
  console.log("Hello, World!");
</script>

Komponen Dasar yang Dipelajari

Berdasarkan panduan Petani Kode, inilah pilar utama yang harus dikuasai:

Variabel

Penyimpan data

Tipe Data

Jenis nilai (teks, angka, dll)

Operator

Operasi matematika/logika

Output

Menampilkan hasil

Alat yang Dibutuhkan

Cukup dua alat sederhana untuk mulai belajar:

  1. Teks Editor: VS Code, Sublime Text, atau Notepad++.
  2. Web Browser: Google Chrome, Firefox, atau Edge (untuk menjalankan kode).
