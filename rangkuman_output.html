Javascript Output

Rangkuman Materi Belajar di Petani Kode

Output adalah cara program menampilkan hasil akhir atau pesan kepada pengguna. Di Javascript, terdapat 4 cara utama untuk menampilkan output.

1. Fungsi console.log()

Digunakan untuk menampilkan teks atau data ke dalam Console Javascript di browser. Biasanya digunakan untuk proses debugging.

console.log("Hello World!");

*Tekan F12 pada browser untuk melihat hasilnya di tab Console.

2. Fungsi alert()

Menampilkan jendela dialog (pop-up) di browser. Berguna untuk memberikan peringatan atau informasi singkat kepada user.

alert("Selamat datang di JS!");

3. Fungsi document.write()

Menuliskan teks atau elemen langsung ke dalam dokumen HTML. Hati-hati, jika dipanggil setelah halaman selesai dimuat, ia bisa menghapus isi seluruh halaman.

document.write("Teks ini muncul di body");

4. Atribut innerHTML

Cara paling fleksibel untuk menampilkan output ke elemen HTML tertentu berdasarkan ID atau class-nya.

let el = document.getElementById("hasil");
el.innerHTML = "Halo dari InnerHTML!";
