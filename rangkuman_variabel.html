Variabel & Tipe Data JS

Rangkuman Materi dari Petani Kode

Variabel adalah "wadah" atau nama yang mewakili sebuah nilai. Di Javascript, variabel bersifat dynamic typing, artinya tipenya bisa berubah-ubah sesuai nilai yang diberikan.

1. Cara Membuat Variabel

Ada tiga kata kunci utama untuk membuat variabel di JS modern:

Lama

var

Cara lama (ES5). Memiliki cakupan function scope.

Modern

let

Cara baru (ES6). Nilainya bisa diubah-ubah. Block scope.

Tetap

const

Untuk nilai tetap (konstanta). Tidak bisa diubah setelah diisi.

2. Aturan Penulisan Nama

  • Tidak boleh diawali dengan angka.
  • Boleh diawali dengan huruf, underscore (_), atau tanda dolar ($).
  • Menggunakan format camelCase (contoh: namaLengkap).
  • Peka terhadap huruf besar/kecil (Case Sensitive).

3. Tipe Data Dasar

var nama = "Petani Kode"; // String (Teks)
var usia = 22;           // Number (Angka)
var tinggi = 172.5;      // Number (Float/Pecahan)
var isLulus = true;      // Boolean (True/False)
var data;                // Undefined (Belum ada isi)

4. Menampilkan Variabel

Gunakan fungsi output untuk melihat isi variabel:

console.log(nama);       // Di Console Browser
document.write(usia);    // Di Halaman Web
alert(tinggi);           // Di Pop-up
